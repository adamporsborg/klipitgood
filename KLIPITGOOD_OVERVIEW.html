README — Digital Product Edition

What gets included in the downloadable kit's README file.

KlipItGood Self-Hosted Kit

Turn any long-form video into hundreds of short-form vertical clips — automatically. No editing skill required. No credit limits. Runs on free infrastructure.

Built by UNSER Media. Powers the hosted KlipItGood service at unser.media.

What This Kit Does

  • Accepts a YouTube link, Drive/Dropbox URL, or direct video upload
  • Transcribes the audio with word-level timestamps (Groq Whisper)
  • Reads the transcript with Claude AI and selects the highest-value clip moments
  • Renders each clip as a vertical 9:16 MP4 with burned-in animated captions (ffmpeg)
  • Uploads clips to Supabase Storage and returns public download links
  • Writes AI social media captions for each clip
  • Sends an email notification when clips are ready

Requirements

Node.js 18+
Python 3.9+
ffmpeg (brew install ffmpeg / apt install ffmpeg)
yt-dlp (brew install yt-dlp)   # only needed for YouTube

API Keys (all have free tiers):
  ANTHROPIC_API_KEY     → console.anthropic.com
  GROQ_API_KEY          → console.groq.com
  SUPABASE_URL          → supabase.com (free project)
  SUPABASE_SERVICE_ROLE_KEY
  RESEND_API_KEY        → resend.com (free 3k emails/mo)

Quick Start (5 Steps)

1. Clone this repo and run:
   npm install && pip install -r worker/requirements.txt

2. Copy .env.example to .env and fill in your API keys

3. Push the Supabase schema:
   psql $SUPABASE_DB_URL < supabase/schema.sql

4. Add your API keys to GitHub Secrets:
   Settings → Secrets → Actions → New repository secret

5. Run your first clip job:
   node worker/klipitgood-worker.mjs --project test-run

Processing a Video

# From a YouTube URL
node worker/klipitgood-worker.mjs --project my-project

# From a local file
FOOTAGE_PATH=/path/to/video.mp4 node worker/klipitgood-worker.mjs --project my-project

# Dry run (transcription + AI analysis only, skip ffmpeg)
node worker/klipitgood-worker.mjs --project my-project --no-render

# Change caption style
node worker/klipitgood-worker.mjs --project my-project --caption-style word-pop

Caption Styles

  • bold-pop — White pill background, large black text, red active word (default)
  • word-pop — Word-by-word reveal, no background
  • minimal — Small white text, bottom-third placement

Automated Processing (GitHub Actions)

The kit includes a GitHub Actions workflow that auto-triggers whenever a new job is queued in Supabase. No server required — runs on GitHub's free Ubuntu runners (~200 free jobs/month).
